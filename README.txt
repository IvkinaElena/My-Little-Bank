Учебный проект на Java 11 + Spring Boot.
Небольшой сайт, который состоит из следующих страниц:
 /index -  список существующих клиентов банка, имеет кнопку для перехода к старнице создания нового клиента;
 /add-customer - форма добавления нового клиента;
 /customer/{id} - список счетов клиента, имеет кнопку для перехода к старнице создания ного счета;
 /add-account/{id} - форма для создания нового счета;
 /transaction/{id} - список транзакций по счету.
